# -End-to-End-ML-Pipeline-for-Customer-Churn-Prediction
Production-ready machine learning pipeline for customer churn prediction using Scikit-learn, with preprocessing, model raining, and hyperparameter tuning. 


##  Objective

* Predict whether a customer will churn or not
* Build a reusable and scalable ML pipeline
* Apply best practices for production deployment



## Methodology

### 🔹 Data Preprocessing

* Missing value handling
* One-hot encoding for categorical variables
* Feature scaling using StandardScaler

### 🔹 Pipeline

* Built using Scikit-learn Pipeline
* Combined preprocessing + model in one flow

### 🔹 Models Used

* Logistic Regression
* Random Forest Classifier

### 🔹 Hyperparameter Tuning

* GridSearchCV with cross-validation
* Optimized model performance

### 🔹 Model Export

* Saved pipeline using joblib
* Ready for reuse and deployment



##  Results

* Random Forest achieved better performance than Logistic Regression
* Hyperparameter tuning improved accuracy
* Pipeline reduced manual preprocessing effort



## Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* Joblib
